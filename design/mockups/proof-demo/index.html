🎯 핵심 컨셉

  • 3-Phase 점진 학습 — 🔵 순서 맞추기 → 🟢 빈칸 채우기 → 🔴 백지 증명
  • GIVEN / THEOREM 명시 — 전제·정의·도구를 학생이 먼저 인지한 뒤 증명 시작
  • 도형 자동 검증 — SVG 좌표로 기하 불변량(평행, 각도, 엇각 등) 프로그래매틱 검증. 잘못된 도형은 자동 거부
  • 즉시 피드백 — Phase 1·2는 실시간 채점, Phase 3만 교사 검토
  • SRS 박스 — Leitner 방식 1→5 박스, 승급 시 복습 간격 확장 (1/3/7/14/30일)
  • 색상 코딩 힌트 — 도형의 엇각 쌍을 같은 색으로 표시해 학생의 시각 추론 유도
